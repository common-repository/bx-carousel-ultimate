== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. From your wordpress dashboard, choose 'Add New' under the 'Plugins' category.
2. Select 'upload' from the set of links at the top of the page (The second links) button
3. From here, browse for the zip file include in your plugin purchase titled 'bx-carousel-ultimate-wordpress.zip' and click the 'install now'
4. Once installation is complete, active the plugin to enable it's features.

After install, the plugin will add a menu custom post called "Bx Carousel Items". Now "Add New" custom post here.

Configuration TinyMCE Features

1. Carousel style : Enable four difference style
2. Carousel ID : Input carousel ID but make sure that it's different from each others
3. Carousel Post Type : Enable two post type here
4. carousel Category : Input carousel category here

Or Basic Usage

<strong>Carousel Style One</strong>
<pre>[bx_carousel_image_only id="different ID" posttype="bx-carousel" category="your Category"]</pre>

<strong>Carousel Style Two</strong>
<pre>[bx_carousel_bx id="different ID" posttype="bx-carousel" category="your Category"]</pre>

<strong>Carousel Style Three</strong>
<pre>[bx_carousel_content id="different ID" posttype="post" category="your Category"]</pre>

<strong>Carousel Style Four</strong>
<pre>[bx_carousel_image_only id="different ID" posttype="post" category="your Category"]</pre>
